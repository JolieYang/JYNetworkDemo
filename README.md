## 介绍
该项目是在学习iOS网络模块的时候进行开发的。学习目标分为两个，一个是对苹果原生API的学习使用：即NSURLConnection以及NSURLSession的基本使用。另一个是AFNetworking的学习，项目中还包含AFNetworking，目前正在读该框架源码中。
